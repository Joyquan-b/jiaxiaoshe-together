## 🔧 自定义调整

### 调整渐变透明度
若觉得背景过暗或过亮，编辑 `index.html` 中的这一行（约第 315 行）：

```css
.section::after {
    background: linear-gradient(135deg, 
        rgba(255,255,255,0.87) 0%, 
        rgba(255,255,255,0.80) 50%, 
        rgba(255,255,255,0.83) 100%);
}
```

修改三个数字的最后一个参数：
- `0.87` = 87% 不透明（可改为 0.75-0.95）
- `0.80` = 80% 不透明
- `0.83` = 83% 不透明

**建议范围**：`0.75` - `0.95`（越低越亮，越高越暗）

### 调整过渡速度
编辑 `.section` 的 `transition` 属性（约第 305 行）：

```css
.section {
    transition: all 0.8s ease-out;  /* 改这个时间 */
}
```

- `0.8s` = 0.8 秒过渡
- 改为 `0.5s` = 更快
- 改为 `1.2s` = 更慢

## ❓ 常见问题

### 图片未显示怎么办？
1. 检查文件名是否正确（如 `2.jpg`）
2. 检查文件格式（jpg/png/webp）
3. 清空浏览器缓存：`Ctrl+Shift+Delete`
4. 确认图片在 `images` 文件夹中

### 首页背景怎么处理？
首页已使用原有的蓝色渐变。若想为首页也添加背景图，编辑 `.hero` 部分：

```css
.hero {
    background-image: url('images/1.jpg');
    background-size: cover;
    background-position: center;
    background-attachment: fixed;
}
```

### 某个section想用纯色背景？
注释掉对应的 `background-image` 行即可，将自动显示备用颜色。
