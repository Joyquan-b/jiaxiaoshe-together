# 项目背景图片指南

## 图片放置说明

请在此文件夹中放入 **7 张学校特色背景图片**，按以下命名规则命名：

### 图片命名和对应的网站板块

| 文件名 | 对应板块 | 建议内容 |
|-------|--------|--------|
| **1.jpg** | 首页 (Hero) | 学校全景、校园风景或学校标志性建筑 |
| **2.jpg** | 项目概述 (Overview) | 项目开展的活动照片或校园相关场景 |
| **3.jpg** | 四维模式 (Mode) | 师生互动、教学场景或学校内部环境 |
| **4.jpg** | 数梦空间 (Shumeng) | 志愿服务活动、团队合作或社区活动 |
| **5.jpg** | 项目成果 (Achievements) | 获奖证书、成果展示或优秀学生活动 |
| **6.jpg** | 团队成员 (Team) | 团队合照、师生合影或工作场景 |
| **7.jpg** | 联系我们 (Contact) | 学校大门、校园俯视图或欢迎场景 |

### 图片要求

- **格式**：jpg、png 或 webp
- **尺寸**：建议最小宽度 1920px，高度 1080px 以上
- **大小**：建议每张 < 2MB（优化后），确保网页加载速度
- **质量**：清晰、色彩鲜艳、能代表学校特色

### 效果说明

✅ **已实现功能**：
1. 每个 section 对应不同的背景图片
2. 背景上方有 **85% 白色半透明渐变叠加**，确保文字清晰可读
3. 背景使用 `background-attachment: fixed` 产生 **视差滚动效果**
4. 各板块之间使用 **0.8s 平滑过渡动画**，避免生硬切换
5. 背景图片自动适应屏幕，自动缩放和居中

### 使用方法

1. 将图片放入此 `images` 文件夹
2. 确保文件名为 `1.jpg`、`2.jpg` ... `7.jpg`
3. 刷新网页，背景自动加载

### 如果图片加载失败

检查：
- ✓ 文件名是否正确（区分大小写）
- ✓ 文件格式是否为 jpg/png/webp
- ✓ 文件是否在 `images` 文件夹中
- ✓ 浏览器缓存是否已清空（Ctrl+Shift+Delete）

### 自定义背景效果

如需调整半透明度或渐变效果，编辑 `index.html` 中的 CSS：

```css
.section::after {
    background: linear-gradient(135deg, rgba(255,255,255,0.85) 0%, rgba(255,255,255,0.75) 50%, rgba(255,255,255,0.80) 100%);
}
```

- 调整 `rgba(255,255,255,0.XX)` 的最后一个数字（0-1 之间）
- 0 = 完全透明，1 = 完全不透明
- 建议范围：0.75 - 0.90

---

**预期效果**：滚动时会看到不同背景逐渐过渡，同时内容卡片始终清晰可见。
